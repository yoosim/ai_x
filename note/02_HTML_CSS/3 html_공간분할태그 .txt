[html - form 태그 / 공간분할 태그]
1. 입력양식(form)태그 
	▶사용자에게 입력 받는 공간을 의미, 입력 양식을 만들 때 사용하는 태그.  
	  예) <form 속성1=속성값 속성2=속성값...>
		<input type="~" name="전달할 파라미터 이름">
 		          	...
	       </form>

	(1) action : 입력 데이터의 전달 위치를 지정
	(2) method : 입력 데이터의 전달 방식을 선택 
	    - get : url에 포함되어 전송(query string 형식)
	    - post : 입력한 데이터가 눈에 보이지 않게 전송 
	(3) button : 버튼을 생성
	(4) checkbox : 체크박스을 생성
	(5) file : 파일 입력 양식을 생성
	(6) hidden : 보이지 않게 합니다.
	(7) image : 이미지 형태를 생성
	(8) password : 비밀번호 입력 양식을 생성
	(9) radio : 라디오 버튼을 생성
	(10) reset : 초기화 버튼을 생성
	(11) submit : 제출 버튼을 생성
	(12) text : 글자 입력 양식을 생성
	(13) autofocus : 포커스 자동 설정
 	(14) placeholder : 각 필드에 대한 입력 형식 힌트제공
	(15) required : 필수 입력 필드
	(16) min, max : 숫자의 범위를 설정
	(17) pattern : 조건을 사용한 일반식 표시
	(18) 사용자 인터페이스를 높이기 위한 label이용

2. 공간분할 태그(레이아웃태그) -div(블록형식의 태그), span(라인형식의 태그)
	(1) div : block 형식으로 공간을 분할. 하나의 블록형식의 태그이므로 개행포함 예)p,ul안의 li
	(2) span : inline 형식으로 공간을 분할 

3. div를 이용한 레이아웃 
4. 시멘틱을 이용한 레이아웃 
	- 시멘틱 구조 태그 
	  (1) header : 해더
	  (2) nav : 네비게이션 
	  (3) aside : 사이드에 위치하는 공간
   	  (4) section : 여러 중심 내용을 감싸는 공간
 	  (5) article : 글자가 많이 들어가는 부분 
 	  (6) footer : 꼬리말
