[html - 글자 태그]
1. HTML   구조 
	(1)<!DOCTYPE html> : 문서의 가장 첫번째 줄에 존재 
	(2) <html  lang=”ko”>  :  HTML 태그의 시작을 알리고 lang  속성입력
		- ko:한국 / en:미국/ ja: 일본 / ru : 러시아 / zh:중국 / de:독일어
	(3) <head>  …  </head> : 이 문서의 정보인 title이나 설정값들인 css, javaScript등의 환경설정에 관한 정보 입력
		- meta : 웹 페이지에 추가 정보를 전달
		- title : 엡 페이지의 제목
		- script : 웹 페이지에 스크립트를 추가
		- link : 웹 페이지에 다른 파일을 추가
		- style : 웹 페이지의 스타일시트를 추가
	(4)<body>  …  </body> : 사용자에게 보이는 웹 문서 실제 정보 
	(5) </html> 
2.글자에 관련된 태그 
	(1) 제목 태그 : H1~H6 (제목 크기)
	(2) 본문 태그 : p (텍스트)
	(3) 개행 태그 : br (줄바꿈)
	(4) 앵커 태그 : a  (다른 페이지나 위치로 하이퍼링크를 연결,href속성 사용)
	(5) 볼드 태그 : b (글자를 굴게 표시 )
	(6) 이탤릭체 : i (기울임 글씨를 강조)
	(7) 위첨자 : sub (글자를 윗첨자로 표시할때 사용 / 예) m<sup>3 =㎥
	(8) 삭제표시 : del (삭제된 텍스트를 의미- 취소선(가운데 선)이 생김)

3. 목록 태그 
	(1) ul : 순서가 없는 목록 태그
	(2) ol : 순서가 있는 목록 태그
	(3) li : 목록 요소

4. 표를 나타내는 태그 - table (표형식의 레이아웃을 만드는 태그로 회원가입폼이나 로그인폼, 게시판폼 등에 다양하게 사용)
	(1) tr : 표 내부의 행 태그
	(2) th : 행 내부의 열. 제목 셀 태그
	(3) td : 행 내부의 열. 일반 셀 태그
	(4) thead : 표의 제목부분 (선택)
	(5) tbody : 표의 본문 (선택)
	(6) tfoot : 표의 하단 (선택)


5. 이미지를 출력하는 태그 - img 
	(1) src : 이미지의 경로 지정
	(2) alt : 이미지가 없을 때 나오는 글자 지정
	(3) width : 이미지의 너비 지정 (css 부분에 설정해주는 경우가 더 많다)
	(4) height : 이미지의 높이 지정 (css 부분에 설정해주는 경우가 더 많다)

6. 오디오 태그 - audio 
	(1) src : 오디오 파일의 경로 지정
	(2) preload : 오디오를 재생하기 전에 모두 불러올지 지정
	(3) autoplay : 오디오를 자동 재생할지 지정 (인터넷 익스플로어에서 작동)
	(4) loop : 오디오를 반복할지 지정
	(5) conrols : 오디오 재생 도구를 출력할지 지정

7. 동영상을 출력하는 태그 - video
	(1) src
	(2) poster : 비디오 준비 중일 때의 이미지 파일 경로 지정
	(3) preload : 비디오를 재생하기 전에 모두 불러올지 지정
	(4) autoplay : 비디오를 자동 재생할지 지정
	(5) loop : 비디오를 반복 재생할지 지정
	(6) controls : 비디오 재생 도구를 출력할지 지정
	(7) width : 비디오의 너비를 지정
	(8) height : 비디오의 높이를 지정
