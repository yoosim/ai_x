1절. 리스트(list) : lst = [타입과 관련없는 0개 이상의 데이터들]
	1) 리스트 요소 추가 
	+ : 두 리스트 연결해서 추가
	* : 리스트를 곱한 수만큼 반복
	append() : 맨 뒤에 요소 추가
	extend() : 맨 뒤에 리스트를 요소별로 추가
	insert(idx, 값) : idx번째 인덱스 위치에 추가(기존 데이터는 shift)
  		  지정된 위치에 추가
	
	2)인덱싱과 요소 갯수
	변수[index] : 인덱스를 이용한 접근 (몇번째에 무슨데이터가 있는지 찾는것) 체크 필요
	index(찾을데이터) : 해당 찾을 데이터가 있는 요소의 index를 반환 (무슨데이터가 몇번째에 있는지 찾는것?)
              ※index는 찾을데이터가 없는 경우 오류 발생
	 ※따라서 in 연산자와 함께 쓴다. 오류나는걸 잡아주기 때문에  
	count(찾을데이터) : 리스트에서 찾을데이터의 갯수 반환

	3)원하는 요소 추출
	3-1.리스트 슬라이싱 : 리스트에서 원하는 index의 데이터를 추출 (인덱스의 범위를 벗어나도 에러 안 남)
	[from:stop:step] / 음수 인덱스 가능 
	3-2.리스트 컴프리헨션 : 리스트에서 원하는 데이터를 추출 
	*컴프리헨션을 못하면 if문법이 많아진다.

	ex) numbers = [1, 3, 5, 9, 10, 12, 13, 20]일때
	result= []
	for data in numbers:
	    if not data%2:   #{}%2의 값이 0일 때 
 	       result.append(data)
	print(result)의 값이 [10,12,20]이 나타남 
	
	컴프리헨션으로 추출하면 
	[data for data in numbers if data%2==0]
	data%2==0 (numbers에서 2를 나눴을 때 0이 되는 숫자는 짝수)를 추출하고 싶을 때

	4)요소 수정하기
	index를 이용해서 요소 수정 가능 number[2] = 222
	슬라이싱을 이용해서 요소 수정 가능 numbers[::2] = ['a','b','c','d','e']

	5)요소 삭제하기 
	pop() : 가장 마지막 인덱스 요소가 반환되고 삭제
	pop(idx) : idx번째 요소가 반환되고 삭제 (뒤에 숫자들은 앞으로 밀려짐)
	remove(지울data) : 리스트안의 data 삭제
	del 변수[idx] : idx번째 요소가 삭
