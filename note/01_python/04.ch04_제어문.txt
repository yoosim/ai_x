[제어문]

▶1절. 조건문 (if)
	if 조건1:
		조건1이 참일 경우 수행할 명령어들
	elif 조건2:
		조건1이 거짓이고, 조건2가 참일 경우 수행할 명령어들
	else:
		모든 조건을 만족하지 않을 때 수행할 명령어들
	
▶2절. 반복문
	2-1 for in
	for 임시변수 in 나열가능한자료:
    		반복할명령어1
		반복할명령어2
 	   	   if 조건:
  	        	   break
	else:
	    나열가능한자료가 다 소요하여 for블록을 다 수행한 후 최종 한번 수행할 명령어
	    for문안에 break를 수행할 경우 else블록은 수행되지 않음

	2-2 while
	while 조건:
	    조건이 참일 경우 반복할 문장들
	else:(break가 있을 경우 생략이 됨)
	    조건이 거짓일 경우 한번 실행할 문장(else구문 선택사항)
