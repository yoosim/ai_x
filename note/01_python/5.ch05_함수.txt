	<지역변수(local ver)>
	특정 함수 안에서만 유효한 변수. 함수가 호출될 때 생성되고 함수가 종료되면 사라짐
	<전역변수(global ver)>
	함수 바깥에서 선언되며 프로그램 어디서나 접근하고 수정할 수 있음

	★변수의 우선순위는 로컬테이블>전역테이블>내장된 이름테이블순서 

▶3절 함수의 매개변수
	- 매개변수에 기본값을 가질 때 : 호출시 선택적으로 명시
		- 매개변수의 기본값이 스칼라 변수일 때 : 기본값이 함수 정의 시점 (*원본값에 영향이 미치지 않음)
		- 매개변수의 기본값이 객체일 때 : 기본값이 함수 호출 시점(*원본값에 영향을 미침)
	- 키워드 인수 : 기본값 갖고 호출시 매개변수 명을 반드시 기입해야 하는 경우
	- 튜플 매개변수 : 가변인수를 구현 *args(함수 정의)
	- 딕셔너리 매개변수 : **kwargs(함수 정의)
	- 튜플인수 언패킹 : 함수 호출시 변수앞에 *
	- 딕셔너리인수 언패킹 : 함수 호출시 변수앞에 **

     * 함수 정의시 매개변수 순서 *
	순서인수 > 튜플인수 > 키워드인수 > 딕셔너리인수

▶4절. 람다식
	funcname = lambda 매개변수들 : 리턴값
	funcname(매개변수들)
	(lambda 매개변수들 : 리턴값)(매개변수들)
      용도 = (1)함수의 매개변수 (2)함수의 return값
         ex1. map(func, 가공될 vector)
         ex2. filter(func, 추출될 vector)
         ex3. sort(key=정렬 기준이 될 데이터를return할 함수, reverse=T/F)
