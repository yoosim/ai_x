[객체지향 프로그래밍]

▶1절. 객체와 클래스
	클래스 : 객체를 만들기 위한 틀
	객체 : 클래스의 인스턴스 
	생성자(constructor)  : (__init__) 객체가 생성될 때 자동으로 호출되는 메서드 
		※생성자 중복정의 불가능, 마지막 정의된 생성자만 유효함
	소멸자(destructor) : (__del__) 객체가 삭제될 때 (메모리에서 제거될 때) 자동으로 호출되는 메서드
	클래스 변수 : 클래스가 메모리에 올라갈 때 생성, 프로그램이 종료되면 소멸, 메소드 영역에 저장
	인스턴스 변수 : 인스턴스가 생성될 때, 인스턴스가 소멸할때 소멸, 힙 영역에 저장


▶2절. 상속과 재정의
	>상속이란? 
	상속을 이용하면 부모클래스의 모든 속성을 자식 클래스로 물려줄 수 있다.
	예시)
	class 클래스명(부모클래스명): 
	calss test01(Fam101): 이라는 가정으로 보면 Fam101의 모든 속성이 test01에게 상속이 된다는 의미 
	calss test01(Fam101):
		pass        pass를 작성한 경우는 추가 삭제 등 변경없이 부모의 속성을 그대로 사용한다는 의미

	override : 함수의 재정의(부모클래스의 함수를 자식클래스에서 다시 정의
           		부모클래스의 함수명과 매개변수 갯수가 동일해야 함
	overload : 함수의 중복정의(같은 이름의 함수를 매개변수 갯수를 달리 해서 정의)
           		파이썬에서 overload는 불가능. c++,java에서 가능 

	>정적 변수란?
	 여러 객체들 사이에서 데이터를 공유하고 싶을때 사용한다. 
 	 static 변수는 공유 변수이다.
	 static 변수 앞에 under score 2개( __ )를 붙이면 클래스명.__클래스명__
	 *클래스명._클래스명__static변수명으로 참조

  		*클래스 변수명은 외부에서 바로 접근이 가능
		*static변수명은 외부에서 바로 접근하는 것을 방지
		*static변수는 클래스 내부에서만 접근하도록 하도록 의도
