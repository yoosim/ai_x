머신러닝/딥러닝 : 데이터의 법칙성을 추출 
 - 지도학습(독립=입력변수와 종속=타켓=출력변수가 모두 주어졌을 경우)
     ①회귀분석 (타겟변수가 연속형) : 연속적인 수치 예측(숫자 예측)
	ex) 집값 예측, 온도 예측, 주가 예측, 5시간 공부하면 80점  등 
	1)선형 회귀(Lineare Regression)
	2)로지스틱 회귀(Logistic Reression)
     ②분류분석(범주를 분류) : 범주(클래스)를 분류 
	(multi-다중(아주춥다,춥다,보통,덥다,아주덥다 등), binary-2개(이진)(춥다,덥다))
	ex) 스팸 메일 분류, 고양이/개 이미지 분류, 질병 유무 예측 
	분류 종류>
	1)이진 분류 : 두가지 클래스중 하나로 분류 (Yes or No) : 스팸/정상 ,질병 있음/없음
	2)다중 분류 : 셋 이상의 클래스 중 하나로 분류  : 숫자 인식 (0~9), 날씨(맑은,흐림,비 등)
	--------> 범주형(카테고리형)데이터를 수치형 데이터로 변환하는 방법
	라벨 인코딩(Label Encoding)  : 각 범주에 정수값 부여, 순서 있음, 순서가 있는 범주에서 사용
	df = LabelEncoder(), fit_transform(data) 
	원핫 인코딩(One-Hot Encoding) : 각 범주를 별도의 열로 표현, 순서 없음, 순서가 없는 범주사용
		to_categorical : 숫자로 된 라벨(정수형 클래스 번호)를 원핫 백터로 변환 / 딥러닝 모델에 라벨을 넣기 위해 사용 
				* numpy 배열로 출력 
		get_dumies(data) : 문자열 혹은 범주형 컬럼을 원핫 인코딩된 데이터 프레임으로 변환 / 머신러닝 전처리 용도로 사용 / DataFrame 이거나 시리즈 
				* pandas DataFrame 으로 출력 
     ③다중 출력 회귀/분류 (Multi-output) : 하나의 입력에 대하여 여러개의 결과 예측 
	ex) 하나의 이미지에서 나이+성별+감정 예측
     ④순서 분류 : 순서가 있는 분류 
	ex) 만족도 (매우 나쁨 < 나쁨 < 보통 < 좋음 < 매우 좋음)

 - 비지도학습(입력변수만)
     ①군집화(Clustering) : 데이터를 비슷한 것들끼리 자동으로 그룹 나누기 (라벨이 없이도 그룹을 나눔)
	1)전체 코드(for문 이용)
	  - 1. data 생성
	  - 2. 랜덤 포인트 2지점 (초기값 임의 생성) : group1과 group2 생성
	  - 3. 포인트 그룹에 따라 랜덤 포인트 이동 
	2)api를 이용한 군집합
	  -sklearn 머신러닝 
	   º 예측모델
	      : 분류, 회귀, 군집분석(n개 그룹으로 알아서 나눔)
		fit(학습), predict(예측)
	   º 변환모델
	      : 전처리 ex.스케일 조정
		fit(학습), transform(변환 함수)
	K-Means 사용 (어떻게 분류되는지는 모름, 컴퓨터가 알아서 분류 함)	
 - 강화학습
